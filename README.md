# Site Web Judo - Projet BUT

## Description du projet

Site web dédié à la découverte du judo, réalisé dans le cadre d'un projet académique de BUT (Bachelor Universitaire de Technologie). Ce site présente l'histoire, les techniques, les champions et les valeurs de cet art martial japonais.
https://trello.com/invite/b/6914abc0058558e3a9ff51f3/ATTI010fa54b89eda0d966a600e7e51483318ABEB5C7/mon-tableau-trello
## Objectifs

- Faire découvrir le judo aux débutants
- Présenter les techniques fondamentales
- Mettre en avant les grands champions
- Partager la passion pour cet art martial technique et satisfaisant

## Fonctionnalités

### Pages principales
 **base.css** : est utiliser pour factoriser le code qui utilise les meme base comme les conteneur et animation de soulignage 

- **Accueil** : Présentation générale du judo avec hero section dynamique
- ** Histoire** : Origines et évolution du judo depuis Jigoro Kano
- **Techniques** : Catalogue des principales techniques (projections, immobilisations, étranglements)
- **Champions** : Profils détaillés des légendes du judo (cartes style TCG)
- ** À propos** : Parcours personnel et motivation du créateur

### Caractéristiques techniques

-  **Mode sombre/clair** : Basculement fluide entre les thèmes
-  **Design responsive** : Compatible mobile, tablette et desktop
-  **Animations CSS** : Effets de survol, transitions et animations
-  **Cartes interactives** : Style Trading Card Game pour les champions
-  **Timeline circulaire** : Présentation chronologique des événements
-  **Galerie** : Affichage dynamique des images
- **Navigation intuitive** : Menu fixe et liens inter-pages

## Technologies utilisées

- **HTML5** : Structure sémantique
- **CSS3** : Styling avancé avec Grid, Flexbox, animations
- **JavaScript** : Interactivité (mode sombre)
- **Design** : Approche mobile-first

## Responsive Design

Le site est optimisé pour tous les écrans :

- **Desktop** : Expérience complète (> 968px)
- **Tablette** : Adaptation des grilles (768px - 968px)
- **Mobile** : Interface simplifiée (< 768px)

##  Fonctionnalités CSS avancées

- **Grid Layout** : Grilles adaptatives pour les cartes
- **Flexbox** : Alignement et disposition flexible
- **Animations** : Keyframes pour effets dynamiques
- **Transitions** : Effets de survol fluides
- **Pseudo-éléments** : Effets visuels (::before, ::after)
- **Gradients** : Dégradés modernes
- **Box-shadow** : Ombres et profondeur
